Overview

A protected breakout for Jetson AGX Orin

The Orin HAT is a breakout board / HAT designed for the NVIDIA Jetson AGX Orin, providing smooth and organized pin connections for prototyping, development, and CAN bus tracing. This board simplifies access to GPIOs, communication buses, and power while ensuring robust protection for both the Jetson and connected peripherals.

All signal lines on the HAT include diodes for reverse current protection, preventing accidental damage from reversed connections or miswiring. The 3.3V rail is supplied directly by the Orin, while the 5V rail is generated by stepping down an external 12V input through a dedicated regulator.

This PCB was designed using Altium Designer and was developed for SC Robotics as a sub-project for the Mars Rover with the assistance of Tyler Le and Cory Hague.

01 — Features

Board Capabilities

  • Breakout / HAT for NVIDIA Jetson AGX Orin
  • Dual voltage setup: 3.3V from Orin, 5V generated from 12V input via step-down regulator
  • IC buffers for clean, stable pin signals
  • Every signal connection is protected by a diode to prevent reverse current
  • CAN communication tracing support
  • Organized breakout pins for GPIO, power, and communication lines
  • Developed for SC Robotics Mars Rover sub-project

02 — Components

Hardware Breakdown

Power Management

Voltage regulator: 12V → 5V for external devices. The 3.3V rail is supplied directly by the Orin.

Signal Protection

IC buffers protect Jetson pins and drive multiple outputs. Diodes provide reverse current protection on all signal lines.

Interface

  • Connectors compatible with Jetson AGX Orin header
  • Decoupling capacitors to stabilize voltage rails
  • Breakout pins: All Jetson GPIOs, power, and communication lines accessible

03 — Wiring & Connections

Pin Assignments

Connector / Pin Function
12V InputPrimary power supply for 5V regulator
5V OutputGenerated from 12V input via step-down regulator
3.3V OutputSupplied directly by the Orin
GPIO / Signal LinesBuffered outputs connected to Jetson pins through diodes for reverse current protection
CAN_H / CAN_LTraced and protected CAN communication lines
Ground (GND)Common ground for Jetson and peripherals

Note: Every signal line passes through a diode, so reverse current from peripherals or miswiring is blocked. Always verify the 12V input is within tolerance.

04 — Usage

Operating Instructions

  • Mount the Orin HAT on the NVIDIA Jetson AGX Orin header
  • Connect a stable 12V power supply to the input (for the 5V rail)
  • Connect peripherals or test circuits to the breakout pins. All signal lines are protected by diodes.
  • Use the buffered GPIOs for clean signal driving and measurement
  • For CAN bus tracing, connect the CAN_H and CAN_L lines; the HAT protects the Jetson pins and allows reliable monitoring
  • Verify voltages at 5V output for any external devices; 3.3V is supplied directly by the Orin

05 — Applications

Use Cases

  • Prototyping with Jetson AGX Orin GPIOs
  • CAN bus development and debugging
  • Sensor integration and robotics projects
  • Educational and research projects with Jetson platforms
