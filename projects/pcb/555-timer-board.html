Overview

A classic timer circuit on custom PCB

This PCB is a simple 555 timer circuit configured in astable mode, designed to blink an LED. It is powered by a 9V battery and includes a single LED with a current-limiting resistor.

This was a project done under IEEE at UC Irvine. The PCB was designed using KiCad. An LTspice simulation file is included for testing and verification.

03 — Usage

Operating Instructions

  • Connect a 9V battery to the PCB
  • The 555 timer begins oscillating in astable mode
  • The LED will blink according to the calculated frequency
  • You can simulate the circuit using the included LTspice file to verify timing and LED behavior

04 — Timing Calculations

Frequency & Duty Cycle

For a 555 timer in astable mode:

Astable Equations Formula 
T_high = 0.693 × (R2 + R3) × C
T_low  = 0.693 × R2 × C
T_total = T_high + T_low = 0.693 × (R3 + 2×R2) × C
f = 1 / T_total
Duty cycle = T_high / T_total

Using Component Values

R2 = 470kΩ, R3 = 1kΩ, C = 1µF

Step 1: Total Period

Period Calculation Math 
T_total = 0.693 × (R3 + 2×R2) × C
T_total = 0.693 × (1,000 + 2×470,000) × 0.000001
T_total ≈ 0.651 seconds

Step 2: Frequency

Frequency Calculation Math 
f = 1 / T_total ≈ 1 / 0.651 ≈ 1.54 Hz

Step 3: Duty Cycle

Duty Cycle Calculation Math 
T_high = 0.693 × (R2 + R3) × C
T_high = 0.693 × (470,000 + 1,000) × 0.000001 ≈ 0.326 s
Duty cycle ≈ T_high / T_total ≈ 0.326 / 0.651 ≈ 50%

Note: R1 (LED resistor) does not affect timing.

05 — Simulation

LTspice Verification

  • You can simulate the circuit using the included LTspice file
  • The OUT pin waveform will correspond to the LED blink pattern
  • Adjust R2, R3, or C to change the frequency and duty cycle as needed
