Overview

A custom PCB built around the ESP32 microcontroller that functions as a wireless joystick remote control. The board integrates analog joystick inputs with the ESP32's onboard Wi-Fi and Bluetooth capabilities, enabling it to transmit control signals wirelessly to a receiving device.

This project combined PCB design with embedded firmware — the board needed to be laid out cleanly around the ESP32's RF antenna clearance requirements, and the firmware needed to read joystick ADC values and transmit them reliably over wireless link.

Key Features

  • ESP32 module with integrated Wi-Fi and Bluetooth for wireless transmission.
  • Analog joystick inputs read via the ESP32's ADC channels.
  • Onboard power regulation for stable 3.3V supply to the microcontroller and peripherals.
  • Compact layout with attention to RF keep-out zone around the ESP32 antenna.
